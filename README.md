# Next.js Dress eShop

in-progress Ecommerce responsive made with `Next.js`, `TypeScript`, `Redux`, `Redux-persist`, `Hooks`, `SCSS` and `BEM`.

## pages

- Home page: /
- Products page: /products
- Product single page: /product/:id
- Cart page: /cart
- Login page: /login
- Register page: /register
- 404 page: /page-not-found
